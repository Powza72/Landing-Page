Projects & Activities

Explore my journey through innovative projects and meaningful activities

Game Compatibility Checker

This web application allows users to check whether their computer can run specific PC games. By selecting a game and entering their system specs (CPU, GPU, RAM, and storage), the site compares their setup against the game's minimum and recommended requirements. It then displays whether the system is compatible or not. The website uses JavaScript to process the logic and a simple, user-friendly interface for input and results.

HTML Tailwind JavaScript

COWBOYDODGE GAME

CowboyDodge Game is a game where players must dodge obstacles and collect coins, with increasing difficulty as the level progresses. The highlight of the game is gesture-based control using hand movements. I used Teachable Machine to train an AI model that recognizes left and right hand raises to control the character. The game was developed with HTML, CSS, and JavaScript, and published via Vercel. This project allowed me to practice both game development and applying AI to a web-based environment.

HTML CSS JavaScript

Fall Detection

In this project, I used the YOLO pre-trained model (yolo11n.pt) from Ultralytics with Python (Flask) to process real-time video and stream MJPEG frames to a web browser. The website was built with HTML, CSS, and JavaScript to display the video. If a fall is detected, the system captures an image and sends an email alert.
